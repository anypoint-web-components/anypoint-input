# \<anypoint-input>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i anypoint-input
```

## Usage
```html
<script type="module">
  import 'anypoint-input/anypoint-input.js';
</script>

<anypoint-input></anypoint-input>
```
